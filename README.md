# Coach Tejaswi Website

A clean, minimalist website for Tejaswi Khanna — writer, coach, and storyteller.

## About

This website showcases the books, writings, and philosophy of Coach Tejaswi. Built with vanilla HTML, CSS, and JavaScript for simplicity and performance.

- **Responsive Design**: Works beautifully on all devices

- **Accessible**: ARIA labels and semantic structure
- **Fast Loading**: Optimized images and minimal dependencies

## Local Development

To run the website locally:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server -p 8000

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

- CSS3 (Custom properties, Grid, Flexbox)

## Contact Form Setup

The contact and subscribe forms use Formspree. To activate:

1. Sign up at [formspree.io](https://formspree.io)
2. Create a new form
3. Replace `your-form-id` in the HTML files with your actual Formspree ID
